Icons

Add icons to your app, only use what you need, separated icons for different themes.

Register icons

App icons can be use anywhere in the app. Icons are registered when needed:

  • Step 1: Define icons sources (.svg, .png, ...), see example.
  • Step 2: Register icons, see example.

Use icons

Try changing to the "Dark theme", the icons will also be changed to adapt.



Sizes

Default icon size is 2x2rem. Add [1-7]x to change icon size. Or set the width and height of the icon tag.
